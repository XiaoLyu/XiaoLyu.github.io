Jekyll+GitHub搭建个人博客

花了好多天，终于算勉强搭好了个人博客。这里选用的是Jekyll。Jekyll, transform your plain text into static websites and blogs.

大概的步骤：

  • 安装Ruby
  • 安装jekyll和bundler
  • 运行jekyll服务器
  • jekyll主题选择
  • 修改jekyll模板
  • GitHub上建立仓库，commit到GitHub上

一、安装jekyll

Install a full Ruby development environment and install Jekyll and bundler gems. 按照以下步骤来，

https://jekyllrb.com/docs/installation/

记住要在命令行里open .bash_profile添加path！

然后创建测试项目，比如说创建一个名为myblog

上面创建项目的时候要等一阵子。如果卡住，可能是有些包没有安装好或者是需要更改默认的source源。

更改默认的source源：

source源有可能变化，注意更改。可以用bundler -v 和 jekyll -v查看它们的版本。

jekyll server启动起来之后，就可以在本地修改之后就查看效果了。可以看到server address：http://127.0.0.1:4000/ （后面kennel有你设置的名称）

二、jekyll主题选择

我是在Jekyll theme挑选jekyll模板的，根据自己需求修改。

jekyll目录结构：

_posts 博客内容
_layouts 网页排版
_sites 最终生成的静态网页
_config.yml 配置信息，这里修改
assets css，images，js

三、配置jekyll, GitPage

  1. GitHub创建仓库：(和用户名相同).github.io
  2. 可以用GitHub Desktop clone 到本地，将修改好的模板覆盖到clone的文件中。
  3. commit到GitHub上，要等一会才能看到效果。
